名詞子句 Noun Clauses

名詞子句是一種句子形式，**整個子句在句中扮演名詞的角色**。 在 TOEIC 中常用於： - 提報告、呈現資訊 - 工作回報 - 表達「知道／相信／確認」類動詞的受詞

一、名詞子句的三大類型

1. 由 that 引導（最常用）

that 可省略但句意不變。
The manager said that the meeting would start at 3 PM.
經理說會議將於下午三點開始。

2. 由 疑問詞（what / when / where / why / how） 引導

用於表達未知資訊。
Please tell me when the shipment will arrive.
請告訴我出貨什麼時候會到達。

3. 由 if / whether 引導（用於 Yes/No 問句）

The supervisor wants to know whether the report is finished.
主管想知道報告是否完成。

二、常見動詞後方常接名詞子句

  • know / understand
  • think / believe
  • say / report / announce
  • explain / describe
  • confirm
We need to confirm that all data has been updated.
我們需要確認所有資料都已更新。

三、TOEIC 常見句型

The report shows that customer satisfaction has increased.
Our team is discussing how we can improve delivery time.
Please let us know whether you can attend the meeting.

四、小練習（非互動題，可搭配課堂講解）

1. The manager will explain ______ the new system works. A) how B) what C) which

建議答案：A
2. We are not sure ______ the package was delivered. A) that B) whether C) why

建議答案：B
3. The CEO mentioned ______ the project would start in June. A) that B) what C) where

建議答案：A
